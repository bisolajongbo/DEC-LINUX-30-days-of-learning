## What I Learned

- Linux Architecture defines how different components of the system interact with each other to manage hardware resources,run applications and provide stable and secure computing environment,Components of Linux OS are Application,Shell,Kernel,Hardware and Utlilities.
Kernel job is to talk to the system hardware and Software and manage the system resources as best as posible.While Shell is a special user program which provides interface for the user to use the os services.I also learnt about the two categories of Shell(Command shell and Graphical shell).The command shell is more powerful than the Graphic Shell,The command shell can work with multiple files with ease.
Also learnt Serveral Available Shell in Linux but Bash which standards for "Bourne Again Shell" is mostly used,we other like CSH(C shell),KSH(Korn shell)etc.
Learnt that $sign in command prompt or Terminal means normal user and  #sign means superuser
I learnt about the Basic Linux Command like ls-means list directory contents,pwd-means print name of current/working directory,cd-means change directory.example cd dirname,man command man - means an interface to the system refernce manuals and clear command-means clear the terminal screen.under command usage i learnt that command lines are case senstive,lower case and upper case letter are not the same.
There is also the Internal commands and External command.
I learnt about the Command Structure,Options in Command,Combing Commands,Multi-line commands.
What is Linux File System is a structured method of storing and organizing data in Linux machine
